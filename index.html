Stock & Shop Management

Como Funciona el algoritmo:
Se plantean dos opciones, para seleccionarlas elegi el prompt de confimacion para no renegar con inputs:

  • Calcular el precio total de una compra: solicita al usuario que ingrese la cantidad de cada producto y devuelve el precio final.
  • Calcular el maximo comprable de cada producto: solicita al usuario el dinero que planea gastar y le devuelve el maximo de cada producto que puede comprar, limitado por el stock. Mi idea es que a futuro no solo de el maximo de cada producto sino que ofrezca algunas combinaciones de prodcutos posibles, para armar combos.
